# План реализации новых режимов тренировки

## Обзор

Добавление трех новых режимов тренировки:
1. **Режим "Слова"** - тренировка на заданное количество слов
2. **Режим "Время"** - тренировка на заданное время
3. **Режим "Цитата"** - тренировка на случайные цитаты

---

## Этап 2: Backend - API для цитат (внешний источник)

### 2.1 Выбор внешнего API

**Для русских цитат**: 
- **Forismatic API** (http://forismatic.com/ru/api/)
  - Endpoint: `http://api.forismatic.com/api/1.0/?method=getQuote&format=json&lang=ru`
  - Бесплатный, без ключа
  - Возвращает: `quoteText`, `quoteAuthor`, `quoteLink`

**Для английских цитат**: 
- **Quotable API** (https://api.quotable.io/)
  - Endpoint: `https://api.quotable.io/random`
  - Бесплатный, без ключа
  - Возвращает: `content`, `author`, `tags`

### 2.2 Создание сервиса для цитат

**Файл**: `backend/app/services/quote_service.py`

**Методы**:
- `get_random_quote(language: str = "ru")` - получить случайную цитату из внешнего API
- Обработка ошибок API (fallback, retry)
- Таймауты (5 секунд)

**Структура ответа**:
```python
class QuoteResponse:
    text: str
    author: str | None
    source: str | None
    language: str
```

### 2.3 Создание роутера для цитат

**Файл**: `backend/app/routers/quotes.py`

**Endpoints**:
- `GET /api/quotes/random?language=ru` - получить случайную цитату из внешнего API

## Этап 5: Frontend - Компонент TypingTrainer

### 5.1 Обновление логики тренировки

**Файл**: `frontend/src/components/TypingTrainer.tsx`

**Изменения**:

#### Режим WORDS (Слова):
- **Текст**: Случайные слова в нижнем регистре из Wikipedia
- **Условие завершения**: `wordCount >= targetWords` (по количеству слов)
- **Подсчет слов**: Разделение по пробелам (`text.split(' ')`)
- **Отображение**: "Слов: X / Y"
- **Генерация текста**: Загрузить текст из Wikipedia, разбить на слова, взять первые `targetWords` слов, объединить в строку

#### Режим TIME (Время):
- **Текст**: Случайные слова в нижнем регистре из Wikipedia (неограниченный поток)
- **Условие завершения**: `timeElapsed >= targetTimeSec`
- **Таймер обратного отсчета**: Показывать оставшееся время
- **Автоматическое завершение**: При достижении времени
- **Генерация текста**: Загрузить длинный текст из Wikipedia, разбить на слова, использовать по мере необходимости

#### Режим QUOTE (Цитата):
- **Текст**: Загружается из API цитат (внешний источник)
- **Условие завершения**: Весь текст введен (как в режиме TEXT)
- **Отображение**: Показывать автора и источник (если есть) под текстом
- **Формат**: "Текст цитаты" — Автор (Источник)

### 5.2 Обновление пропсов TypingTrainer

**Новые пропсы**:
```typescript
interface TypingTrainerProps {
  text: string
  mode: TrainingMode
  textId?: string
  targetWords?: number      // Для режима WORDS (10, 25, 50, 75, 100)
  targetTimeSec?: number     // Для режима TIME (15, 30, 60, 120)
  quoteAuthor?: string      // Для режима QUOTE (отображение)
  quoteSource?: string       // Для режима QUOTE (отображение)
  onFinish: (stats: TrainingStats) => void
}
```

## Этап 6: Frontend - Страницы новых режимов

### 6.1 Страница "Тренировка: Слова"

**Файл**: `frontend/src/pages/TrainWordsPage.tsx`

**Функционал**:
- Выбор количества слов: **10, 25, 50, 75, 100** (кнопки)
- Источник текста: **только Wikipedia** (автоматически)
- При выборе количества слов:
  - Загрузить текст из Wikipedia
  - Разбить на слова
  - Взять первые N слов
  - Объединить в строку (все слова в нижнем регистре)
- Запуск тренировки с параметром `targetWords`

### 6.2 Страница "Тренировка: Время"

**Файл**: `frontend/src/pages/TrainTimePage.tsx`

- Источник текста: **только Wikipedia** (автоматически)
- При выборе времени:
  - Загрузить длинный текст из Wikipedia (минимум 2000 символов)
  - Разбить на слова
  - Все слова в нижнем регистре
  - Использовать по мере необходимости (текст не заканчивается)
- Запуск тренировки с параметром `targetTimeSec`
- Таймер обратного отсчета в UI

### 6.3 Страница "Тренировка: Цитата"

**Файл**: `frontend/src/pages/TrainQuotePage.tsx`

**Функционал**:

- Выбор языка: **Русский / English** (переключатель)
- Отображение:
  - Текст цитаты (для тренировки)
  - Автор (если есть) - под текстом
  - Источник (если есть) - под автором
- Запуск тренировки с текстом цитаты

## Этап 8: Backend - Генерация случайных слов из Wikipedia

### 8.1 Обновление WikipediaService

**Файл**: `backend/app/services/wikipedia_service.py`

**Новые методы**:
- `get_random_words(language: str = "ru", word_count: int = 100)` - получить случайные слова

  - Очистить текст (убрать HTML, ссылки, лишние символы)

  - Загрузить статью из Wikipedia
  - Очистить текст
  - Разбить на слова

  - Объединить в строку (минимум 2000 символов)

## Примеры использования:

### Режим WORDS:
```
Пользователь выбирает: 50 слов
→ Загружается текст из Wikipedia (API)
→ Текст разбивается на слова
→ Берется первые 50 слов
→ Все слова переводятся в нижний регистр
→ Объединяются в строку: "слово1 слово2 ... слово50"
→ Тренировка начинается
→ Завершается после ввода 50 слов (подсчет по пробелам)
```

### Режим TIME:
```
Пользователь выбирает: 60 секунд (1 минута)
→ Загружается длинный текст из Wikipedia (минимум 2000 символов)
→ Текст разбивается на слова
→ Все слова переводятся в нижний регистр
→ Тренировка начинается
→ Таймер обратного отсчета: 60 → 59 → ... → 0
→ Автоматически завершается через 60 секунд
→ Текст не заканчивается (используется по мере необходимости)
```

### Режим QUOTE:
```
Пользователь выбирает язык: Русский
→ Нажимает "Новая цитата"
→ Загружается случайная цитата из Forismatic API
→ Отображается:
   - Текст цитаты (для тренировки)
   - Автор (если есть)
   - Источник (если есть)
→ Тренировка начинается
→ Завершается после ввода всей цитаты
```
